# Algorithm
올바른 알고리즘: 어떠한 경우에도 실행 결과가 똑같이 나오는 것.
## 목표
백준 티어 골드 달성하기

600문제 이상 풀기 (기초 제외)

## 규칙
- 매일 최소 1개 이상의 알고리즘 문제를 풀고 풀이과정을 커밋합니다.
- 30분 ~ 1시간 정도로 시간제한을 두고 문제를 풉니다.
- 시간내에 풀지 못하면 답을 보고, 정답코드에서 몰랐던 부분을 공부합니다.
- 풀지 못한 문제는 1 ~ 2일 뒤에 다시 풀어봅니다.
